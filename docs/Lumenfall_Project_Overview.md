# Lumenfall – Project Overview

## What This Document Is
A high-level explanation of the Lumenfall project for any new developer or AI entering
the repository.

## What Lumenfall Is
- A hybrid ARPG + TCG game.
- Systems include packs, rarities, essence, card instances, Sharenbaal, regions, classes,
  elements, currencies, and endgame narrative ("The Guide").

## Core Components
- Mechanical specs (YAML)
- Lore and experience specs (Markdown)
- Code implementation (Kepler)
- System design (Neotus)
- Project memory (Atlas)
