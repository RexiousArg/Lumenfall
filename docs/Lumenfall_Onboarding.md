# Lumenfall Onboarding Guide

Welcome to the Lumenfall project.

## What to Read First
1. Lumenfall_Project_Overview.md
2. Lumenfall_Spec_Index.yaml
3. Lumenfall_Devlog.md (latest entries)

## How Specs Are Organized
- mechanics → YAML
- lore → Markdown

## How to Contribute Safely
- Never delete files; integrate changes and keep append-only logs where indicated.
- When adding or updating specs, bump spec versions, update the Spec Index, log in the Devlog, and add TODO items if work is required.
- Reload `/docs/specs/` and core docs at the start of each session to maintain continuity.
